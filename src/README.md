# src/ - フロントエンド（React）

React 19 + Tailwind CSS 4 + Viteで構築されたシングルページアプリケーションです。

## ページ構成

**Home.tsx** はランディングページで、システムの概要と主要機能を紹介します。管理画面へのリンクとデモフォームへのリンクを含みます。

**PublicForm.tsx** は公開フォームページで、URLパラメータのスラッグとフォームタイプに基づいて動的にフォームを表示します。サロンごとに設定されたデザインテーマが自動的に適用され、送信データはバックエンドを経由してLark BASEに保存されます。4種類のフォーム（新規顧客情報、カルテデータ、月間目標、年間目標）をタブで切り替えられます。

**Dashboard.tsx** は管理画面で、サロンの作成・管理、デザインテーマの選択、Lark API認証情報の設定、送信履歴の確認が行えます。

## APIクライアント

`lib/api.ts` はfetchベースのシンプルなAPIクライアントです。tRPCの代わりにREST APIを直接呼び出す設計で、Cloudflare Pages Functions環境に最適化されています。開発時はViteのプロキシ設定により `/api/*` へのリクエストがバックエンドに転送されます。
