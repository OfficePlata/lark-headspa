# Shared

フロントエンドとバックエンドの両方で使用される共有コードです。型定義、定数、デザインテーマ設定を含みます。

---

## ファイル構成

| ファイル | 説明 |
|----------|------|
| `themes.ts` | デザインテーマ定義とデフォルトフォーム設定 |
| `const.ts` | 共有定数（Cookie 名等） |
| `types.ts` | 共有型定義 |
| `_core/` | フレームワーク基盤の共有コード。編集不要 |

---

## themes.ts

デザインテーマとデフォルトフォーム設定を定義するファイルです。

### ThemeConfig インターフェース

各テーマは `ThemeConfig` インターフェースに従って定義されています。カラーパレット（16 色）、フォント設定（見出し・本文）、角丸サイズ、シャドウの 4 カテゴリで構成されます。

### 定義済みテーマ

| テーマ ID | 日本語名 | 概要 |
|-----------|----------|------|
| `calmer` | カルメ | ベージュ × ゴールドの癒し系デザイン |
| `natural` | ナチュラル | グリーン × アースカラーの自然派デザイン |
| `elegant` | エレガント | ネイビー × ゴールドの高級感あるデザイン |
| `fresh` | フレッシュ | ライトブルー × ホワイトの清潔感あるデザイン |
| `sakura` | サクラ | ピンク × ローズの華やかなデザイン |

### DEFAULT_FORM_CONFIGS

4 種類のフォームのデフォルトフィールド構成を定義しています。

| フォームタイプ | タイトル | フィールド数 |
|---------------|----------|-------------|
| `customer` | 新規顧客情報入力フォーム | 8 |
| `karte` | 顧客管理情報入力フォーム | 12 |
| `monthly_goal` | 月間目標入力フォーム | 4 |
| `yearly_goal` | 年間目標入力フォーム | 4 |

各フィールドは `fieldName`（Lark BASE のフィールド名に対応）、`fieldLabel`（表示ラベル）、`fieldType`（入力タイプ）、`placeholder`、`isRequired`、`options`（セレクトボックス用）で構成されます。

### エクスポート

| 名前 | 種別 | 説明 |
|------|------|------|
| `THEMES` | Record | テーマ ID をキーとしたテーマ設定マップ |
| `THEME_LIST` | Array | テーマ設定の配列 |
| `getTheme(id)` | Function | テーマ ID からテーマ設定を取得（未定義時は calmer） |
| `DEFAULT_FORM_CONFIGS` | Object | フォームタイプごとのデフォルト設定 |
| `FORM_TYPES` | Array | フォームタイプの文字列配列 |
| `FormType` | Type | フォームタイプのユニオン型 |
